Installation
############

The easiest way to install geovar is via ``pip`` or ``pip3`` 

.. code-block:: bash

  pip install git+https://github.com/aabiddanda/geovar

You can also install geovar locally by:

.. code-block:: bash

  git clone https://github.com/aabiddanda/geovar
  cd geovar/
  pip install .
